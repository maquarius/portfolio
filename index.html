Projects

Fitness Portal

My submission for the final project of my front-end development course. It is a portal for a personal trainer, which shows the trainings and customers. The data comes from heroku, so it takes a while to see the result! Allows the data to be changed by using CRUD operations, the back-end is provided by the teacher for this course. Made with react.

University portal

My own university portal containing all of the relevant links. It also allows me to set one goal at a time so that i focus on it better. The name and goal is stored locallly so you can try it out yourself! The background changes depending on the time. Made with JS, HTML and CSS.

Jammming

This is my implementation of the Codecademy Full-Stack devlepment project. It uses the Spotify API to authenticate and fetch songs, artist or albums based on the seach criteria. Those songs can be added to a playlist that is made in the app and eventually saved to your own spotify account! Uses React. Although i followed a guide while making, i did deviate from it by implementing async/await instead of promises.

Restaurant loader

Using the restaurants API from Wolt. It shows the restaurants in Helsinki. I used this to practice showing an intermidiate loading screen before the images are loaded. Made with React.

Moodbeats

Moodbeats was an application I designed together with my team in the Nordic Health Hackathon. Our task was to improve overall health through the use of an application. We came up with Moodbeats, which detects which music has which effect on your heart rate. During this hackathon I was responsible for the final presentation and I had a shared responsibility for the design with Adobe XD.
